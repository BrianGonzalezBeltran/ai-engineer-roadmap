BrainIt is Brian González's AI Engineering portfolio and platform, hosted at brainit.run. It runs on an Oracle Cloud free-tier server with 4 ARM OCPUs, 24GB RAM, and Ubuntu 24.04. The server hosts multiple services including n8n for workflow automation, FastAPI applications, and Ollama for local LLM inference. Current projects include an AI-powered chat API using phi3:mini and a RAG system for document-based question answering. The tech stack includes Python, FastAPI, Docker, ChromaDB for vector storage, and Ollama for embeddings and language model inference. Brian's goal is to build a portfolio of 3-4 live AI projects deployed on this infrastructure to demonstrate production-ready AI Engineering skills.
